Как установить

  1. Нажми Скачать APK и перекинь файл на Android-телефон.
  2. Открой файл и разреши установку из неизвестных источников.
  3. Запусти приложение и разреши доступ к камере.
  4. Наведи на текст — перевод появится поверх него.
Если кнопка ведёт на ошибку 404 — значит сборка ещё идёт. Подожди пару минут и обнови страницу. Кнопка всегда ведёт на самую свежую сборку.

Что умеет

  1. Языки перевода: English, Русский, Español, Français, Deutsch, Italiano, Português, 中文, 日本語, 한국어.
  2. Распознавание с камеры: латиница, 中文, 日本語, 한국어, देवनागरी (переключается в приложении).
Исходный код и сборки — на GitHub. Собрано автоматически через GitHub Actions.
