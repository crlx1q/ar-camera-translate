# AR Camera Translate

Реал-тайм переводчик с камеры на Flutter. Наводишь камеру на текст — перевод рисуется прямо поверх оригинала и **держится на нём**, поворачиваясь вместе с надписью, когда ты двигаешь телефон.

## Возможности
- 📷 Реалтайм-распознавание текста с камеры (Google ML Kit, офлайн)
- 🌐 Автоопределение языка исходного текста
- 🔤 Офлайн-перевод на устройстве, 10 языков, без API-ключей
- 🎯 Оверлей перевода привязан к позиции и углу наклона оригинального текста

## Языки перевода
English, Русский, Español, Français, Deutsch, Italiano, Português, 中文, 日本語, 한국어.

## Как получить APK
Сборка происходит автоматически в GitHub Actions при каждом push в `main`.

1. Открой вкладку **Actions** → последний запуск workflow **Build APK**.
2. Скачай артефакт **ar-camera-translate-apk** внизу страницы запуска, либо возьми `.apk` из раздела **Releases**.
3. Перекинь `app-release.apk` на Android-телефон и установи (включи «Установка из неизвестных источников»).

Можно запустить сборку вручную: **Actions → Build APK → Run workflow**.

## Ограничения OCR
Бесплатный офлайн-движок ML Kit распознаёт с камеры только скрипты: **латиница, 中文, 日本語, 한국어, देवनागरी**. Кириллица и арабский с камеры **не распознаются** (перевод *на* русский при этом работает). Скрипт распознавания переключается кнопкой в правом верхнем углу.
